Бұл файл **JetDelivery** жобасының ішкі механизмдерін, деректер алмасу логикасын және визуалды компоненттер жүйесін егжей-тегжейлі талдауға арналған. Жобаның негізгі мақсаты — Алматы қаласы бойынша жылдам тамақ жеткізу процесін автоматтандыру.

## Жүйелік Функциялар (Feature Breakdown)

| Модуль | Компоненттер / Парақшалар | Сипаттамасы мен Логикасы |
| :--- | :--- | :--- |
| **Авторизация** | `auth/login`, `auth/register` | `NextAuth.js` арқылы жүзеге асырылған қауіпсіз кіру жүйесі. Құпия сөздер `bcryptjs` арқылы хэштеліп сақталады. |
| **Негізгі Бет** | `page.tsx` | Динамикалық Hero секциясы, танымал тауарлар қосымшасы және тапсырыс беру қадамдары. Көз тартарлық анимациялармен оңтайландырылған. |
| **Каталог** | `catalog/page.tsx` | Тауарларды сатылым бағасы, санаты және қоймадағы қолжетімділігі бойынша нақты уақытта (real-time) сүзгілеу. |
| **Себет** | `CartDrawer.tsx` | Пайдаланушының құрылғысында (`LocalStorage`) сақталатын себет жүйесі. Кедергісіз кезөкелген беттен ашылып-жабылады. |
| **Төлем (Checkout)** | `checkout/page.tsx` | Тұтынушы мен мекенжай мәліметтерін жинап, дерекқорға жаңа тапсырыс жоғары қауіпсіздікпен тіркеледі. |
| **Қадағалау** | `orders/page.tsx` | Тапсырыстың 4 түрлі күйін визуализациялау өзегі: бақылау сызығы (Tracker Bar). Әр тапсырыс жеке карточка арқылы бақыланады. |
| **Әкімші Панелі** | `admin/page.tsx` | Оқшауланған басқару орталығы. Клиенттер мен кірістер статистикасы, тауарларды интерактивті түрде (inline) өзгерту немесе толығымен өшіру мүмкіндігі орнатылған. |

## Базалық API Бағыттары (REST Endpoints)

Жоба ішкі `Next.js App Router` арқылы оқшауланған және қорғалған API жүйесіне ие. Әр сұраныс барысында сессия (session) міндетті түрде тексеріледі. 

| Мекенжай (Route) | Әдіс | Арналуы | Рұқсат деңгейі |
| :--- | :--- | :--- | :--- |
| `/api/categories` | `GET`, `POST` | Санаттар тізімін алу немесе жаңа санат қосу | Оқу: Бәрі |
| `/api/products` | `GET`, `POST` | Каталогты алу немесе тауар қосу | Оқу: Бәрі |
| `/api/products/[id]`| `PATCH`, `DELETE`| Тауардың бағасын, суретін, санын өзгерту / Өшіру | Тек Әкімші (Admin) |
| `/api/orders` | `POST`, `GET` | Тапсырысты рәсімдеу және пайдаланушы тапсырыстарын алу | Авторизацияланған қолданушы |
| `/api/orders/[id]` | `PATCH` | Тапсырыстың күйін (status) жаңарту (`pending` -> `delivered`) | Тек Әкімші (Admin) |
| `/api/admin/users` | `GET` | Барлық пайдаланушылар тізімі мен олардың статус-рөлдері | Тек Әкімші (Admin) |
| `/api/admin/users/[id]`| `PATCH` | Пайдаланушы рөлін өзгерту (мысалы: `customer` -> `admin`) | Тек Әкімші (Admin) |

## Дизайн және Дизайн Жүйесі (Design System)

Бұл жоба ешқандай CSS-фреймворктерсіз (Tailwind немесе Bootstrap), таза `Vanilla CSS` стилизациясы арқылы жасалған. Басты мақсат — жобаның бірегейлігі мен жоғары жылдамдығы.

1.  **Түстер палитрасы:**
    *   `--brand`: Жарқын сары (Amber) — Жылдамдық пен тәбетті шақыру үшін.
    *   `--bg`: Қатты қара (Solid Charcoal) — Премиум көрініс пен контраст үшін.
2.  **Типография:** 
    *   `Bricolage Grotesque`: Үлкен тақырыптар үшін, заманауи геометрия.
    *   `Plus Jakarta Sans`: Негізгі мәтін мен ұсақ бөлшектер үшін, таза оқылымдылық.
3.  **Анимациялар:** Барлық өтулер `cubic-bezier(0.22, 1, 0.36, 1)` логикасымен ұйымдастырылған. Карточкалардың қалқып шығуы (rise) мен микро-интеракциялар максималды бірқалыпты жұмыс істейді.
